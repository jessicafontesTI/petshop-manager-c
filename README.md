# 🐾 Pet Management System

Sistema de cadastro e gerenciamento de pets desenvolvido em linguagem **C**, utilizando o padrão **CRUD (Create, Read, Update e Delete)** para organizar informações de animais e seus respectivos tutores.

Este projeto foi desenvolvido com foco na aplicação prática dos conceitos de lógica de programação, estruturas de dados e organização de código, simulando uma solução para pequenos pet shops e clínicas veterinárias.

---

## 📌 Sobre o Projeto

Muitos pequenos estabelecimentos ainda utilizam planilhas ou registros em papel para controlar informações de clientes e animais. Esse método pode gerar dificuldades na localização de registros, duplicidade de informações e maior possibilidade de erros.

Com o objetivo de resolver esse problema, foi desenvolvido um sistema em linguagem C que permite cadastrar, consultar, editar, excluir e localizar registros de pets utilizando o CPF do tutor como identificador.

---

## 🚀 Funcionalidades

- ✅ Cadastro de pets e tutores
- ✅ Listagem de todos os registros
- ✅ Busca de pets pelo CPF do tutor
- ✅ Edição de cadastros
- ✅ Exclusão de registros
- ✅ Validação de CPF (11 dígitos numéricos)
- ✅ Validação de idade e peso
- ✅ Interface em modo texto (Console)

## ⚙️ Como Executar

Clone o repositório:

```bash
git clone https://github.com/jessicafontesTI/petshop-manager-c.git
```

Acesse a pasta do projeto:

```bash
cd petshop-manager-c/src
```

Compile o programa:

```bash
gcc main.c -o petshop
```

Execute:

```bash
./petshop
```
